## 개발 표준
### 언어 및 개발환경
- 게임 엔진: Unity 6
- 스크립트: C# .NET Standard 2.1
- 3D 모델링: Blender
- 형상관리: Github (한글로 작성)

### 프로젝트 폴더
```
Assets
├── Art
│   ├── Material  # 재질 에셋 
│   ├── Model     # 모델링 파일 e.g. fbx
│   ├── Prefab    # 오브젝트 프리팹
│   ├── Sound     # 음향 효과
│   ├── Texture   # 텍스처 에셋
│   └── UI        # 인터페이스, 2D Sprite
│
├── Scenes
│
└── Script
    ├── asdf      # 스크립트
    └── qwer      # 스크립트
```
